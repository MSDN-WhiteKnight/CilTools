Syntax
public class CilGraphNodeMutable : CilGraphNode
Remarks

Graph nodes are exposed as immutable CilGraphNode objects by API of this library, such as GetGraph(MethodBase) method, because usually you don't need to alter their properties. However, these nodes are created as mutable objects and their actual runtime type is CilGraphNodeMutable; you can cast them to that type if you need to set their properties.
